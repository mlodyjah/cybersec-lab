# DVWA Local Lab Setup

## Goal
Create a local vulnerable web application environment using Docker.

## Technologies
- Docker
- Docker Compose
- DVWA

## Steps performed
- installed Docker Desktop
- configured Docker storage on external SSD
- deployed DVWA container
- exposed web application on localhost:8080

## Skills learned
- Docker basics
- container deployment
- local web environments

## Enumeration

### Docker
- checked running containers
- analyzed container logs
- accessed container shell using docker exec

### Nmap
- scanned localhost services
- identified Apache web server
- detected exposed HTTP port
